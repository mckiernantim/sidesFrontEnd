# Last Looks Component Flow Documentation

This document outlines the flow and interaction between the key components of the Last Looks feature, which allows users to view and edit PDF documents in the application.

## Component Structure

The Last Looks feature consists of the following key components:

1. **LastLooksComponent** - Parent component that manages the overall document state
2. **LastLooksPageComponent** - Child component that handles individual page rendering and interactions
3. **DragDropService** - Service that provides drag and drop functionality
